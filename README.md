# InfoMap - ニュース情報空間可視化プロトタイプ

LLMとナレッジグラフを活用した、多様な情報に出会えるニュースサイト拡張機能のプロトタイプです。

## 概要

ニュース記事を2Dマップ上に可視化し、ユーザーの情報偏向を直感的に把握できるようにします。フィルターバブルやエコーチェンバーの問題に対処することを目的としています。

## 機能

- **2Dマップ可視化**: 記事をトピック別（政治、経済、社会、国際、テクノロジー、環境）に色分けして表示
- **インタラクション**: ホバーでツールチップ、クリックで記事選択、ズーム＆パン対応
- **おすすめ記事**: 異なるトピック・未閲覧トピックの記事を優先的に推薦
- **多様性スコア**: シャノンエントロピーで閲覧の多様性を0-100で表示
- **閲覧履歴**: 読んだ記事の履歴を追跡

## 起動方法

```bash
# ローカルサーバーを起動
python3 -m http.server 8080

# ブラウザで開く
open http://localhost:8080
```

## 技術スタック

- HTML/CSS/JavaScript
- D3.js (データ可視化)
